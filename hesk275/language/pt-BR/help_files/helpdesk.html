Pasta de administração

Nome da pasta onde os arquivos de administração estão localizados.

A pasta precisa existir dentro da pasta principal do HESK. Na maioria dos sistemas os nomes diferenciam MaIúScUlAs E mInÚsCuLaS (CaSe SeNSiTiVe).

Caracteres permitidos (outros serão removidos): a-z A-Z 0-9 _ -

Padrão: admin

Caracteres permitidos (outros serão removidos): a-z A-Z 0-9 _ -

Padrão: attachments

Fechar chamados automaticamente

Se um chamado não tem atividade do cliente por X dias é fechado automaticamente. Defina como 0 para desabilitar o fechamento automático.

Máximo abertos

Um número máximo de chamados abertos que um cliente pode ter. Quando este limite é alcançado o cliente não poderá enviar novos chamados até que os existentes sejam resolvidos. Afeta apenas chamados enviados pelo formulário on-line.

Defina como 0 para desabilitar o limite máximo de chamados abertos.

Permitir login automático

Se definido como SIM a equipe terá uma opção para logar automaticamente em suas contas sempre que abrirem o painel administrativo do HESK em seus computadores. Se desabilitado o HESK será apenas capaz de lembrar usuários, não senhas.

Atribuição automática de chamados

Os chamados serão automaticamente atribuídos a um membro da equipe quando enviados.

Diversos fatores são levados em conta ao atribuir chamados automaticamente, como a categoria do chamado e o número de chamados abertos atribuídos atualmente.

Require owner

If set to ON tickets will need to be assigned before staff can reply to them.

This helps prevent multiple staff working on the same ticket by mistake as it forces them to assign the ticket to themselves before starting to work on it.

Reabrir chamados

Defina como LIGADO para permitir aos clientes reabrir um chamado uma vez que foi fechado. Quando definido como DESLIGADO uma vez fechados os clientes terão de abrir um novo chamado.

Avaliação de resposta

Se definido como LIGADO os clientes poderão marcar as respostas da equipe como Útil ou Não foi útil. Este feedback dos clientes será usado para avaliar a equipe.

Prioridade do cliente

Se definido como LIGADO os clientes poderão definir a prioridade/urgência para seus chamados variando de Baixa à Alta. Se definido como DESLIGADO todos os chamados serão enviados com a prioridade padrão (Baixa) e apenas a equipe poderá alterar o nível de prioridade para um maior.

Identificação sequencial

Se habilitado todos os chamados também exibirão um número de identificação sequencial. Isto deve permitir uma comunicação mais fácil entre cliente-equipe sobre um chamado (dizendo identificação do chamado 45 ao invés de identificação do chamado GHS-G5E-A6T8). Entretanto, apenas a equipe poderá ver os chamados pela numeração sequencial.

ReCaptcha

ReCaptcha é um serviço anti-bot gratuito do Google.

Para usar o ReCaptcha você precisará:

  1. Cadastrar-se no ReCaptcha
  2. Clicar em Get reCAPTCHA
  3. Informe os detalhes do seu site no formulário Register a new site
  4. Copie a Site key nas configurações do HESK
  5. Copie a Secret key nas configurações do HESK
  6. Salve as configurações do HESK
  7. Tente enviar um novo chamado através da interface do cliente para verificar se o reCAPTCHA está funcionando.

Usar pergunta anti-SPAM

Alterna o uso da pergunta de segurança anti-SPAM. Esta é uma maneira muito simples e ainda assim efetiva de combater spammers (exceto se eles decidirem atacar seu site especificamente, mas deve bloquear todos os spambots navegando pela internet). A ideia é criar uma pergunta única que apenas humanos possam responder. Você pode usar código HTML na pergunta. As respostas não diferenciam MaIúScUlaS De MiNúScUlaS (não são CaSe SeNSiTiVe). Alguns exemplos:

-> Pergunta -> Resposta
Qual a cor da água? Azul
Qual é o próximo número após o quatro? (use apenas números) 5
Digite o código de acesso ABCDE aqui: ABCDE
Código de acesso: (encontre-o no inferior de nossa página de links) Somecode

Atenção! NÃO use estes exemplos, use uma pergunta única que será usada apenas em seu site!

Limite de tentativas de login & Tempo de banimento (minutos)

Este recurso limita ataques de força bruta tanto para entrar no HESK quanto para visualizar chamados.

Após o número de Limite de tentativas de login ao visualizar um chamado ou entrar no HESK o endereço IP será banido por Tempo de banimento (minutos).

Para desabilitar o limite de tentativas falhas de login, defina o valor em 0 (NÃO recomendado!).

Frames

Habilite para impedir o HESK de carregar em frames em sites de terceiros. Isto ajuda a combater tentativas de "clickjacking".

Quando habilitado o HESK enviará um cabeçalho HTTP X-Frame-Options: SAMEORIGIN ao navegador.

Se você deseja exibir o HESK dentro de um frame em um site em um domínio diferente do qual o HESK está instalado desabilite este recurso.

SSL

Você pode forçar o HESK a usar conexões seguras (https versus http).

Para gerenciar esta configuração primeiro abra manualmente o arquivo admin_settings.php no seu navegador usando https://, por exemplo:

https://exemplo.com.br/hesk/admin/manage_settings.php

Isto é para assegurar que o SSL está configurado adequadamente e funcionando no servidor antes de configurar o HESK para forçar seu uso.

» Anexos

Usar anexos

Selecione SIM para habilitar anexo de arquivos. Se habilitado os clientes e a equipe de suporte terão a opção de anexar arquivos aos chamados.

OBSERVAÇÃO: Se e como os anexos de arquivos funcionarão dependerá das configurações do seu servidor, por favor consulte o readme.html para considerações e possíveis limitações de anexos de arquivos em seu servidor.

Número por post

Número de arquivos anexos permitidos por post. O limite ideal é entre 1 a 4. Esta configuração só está disponível se "Usar anexos" está definido como SIM.<

 

Fechar janela

 

Tamanho máximo do arquivo

Tamanho máximo de arquivos anexados. Observe que o tamanho máximo de envio de arquivo é limitado pelas configurações do seu servidor, por favor consulte o readme.html para maiores informações sobre limites de anexo de arquivos. Esta configuração só está disponível se "Usar anexos" está definido como SIM.

Tipos de arquivos permitidos

Extensões de arquivos permitidas para anexar arquivos. Para adicionar mais extensões use o mesmo formato .ext e os separe com uma vírgula. Esta configuração só está disponível se "Usar anexos" está definido como SIM.
