Habilitar base de conhecimento

Use esta configuração para ligar ou desligar o recurso de base de conhecimento do HESK.

As opções disponíveis são:

  • SIM, habilitar base de conhecimento
    Habilita o recurso de base de conhecimento do HESK
     
  • SIM, usar o HESK apenas como base de conhecimento (desabilita a central de suporte)
    Habilita a base de conhecimento, mas desabilita o recurso de central de suporte (enviar novos chamados). Use esta opção se você deseja usar o HESK como uma base de conhecimento, não uma central de suporte completa.

    Observe que chamados ainda podem ser enviados pela equipe ou pela fila por e-mail/busca em POP3 se habilitados. Os clientes podem ver e responder aos chamados.
     
  • NÃO, desabilitar base de conhecimento
    Desabilita o recurso de base de conhecimento do HESK

Editor WYSIWYG

Isto habilitará um editor do tipo What You See Is What You Get (O Que Você Vê É O Que Você Tem) (um editor de texto rico) para os artigos da base de conhecimento ao invés de um editor de texto simples.

Sugerir artigos da BC

Se definido como SIM o Hesk verificará o assunto e a mensagem do chamado com artigos da base de conhecimento e recomendará aos clientes que leiam os artigos correspondentes antes de enviar um novo chamado.

Habilitar avaliações na BC

Defina como SIM para permitir aos clientes avaliar artigos na base de conhecimento como Útil ou Não foi útil

Habilitar pesquisa na BC

Habilitar este recurso exibirá um formulário de pesquisa no topo da interface do cliente na central de suporte e permitirá que eles usem o formulário de pesquisa para pesquisar sua base de conhecimento.

Você pode escolher exibir uma caixa de pesquisa pequena (localizada no canto superior direito) ou uma maior (mais visível, cobrindo toda a largura da página).

Exibir visualizações do artigo

Defina como SIM para exibir publicamente o número de visualizações do artigo da base de conhecimento (visualizações são sempre visíveis no painel de controle da equipe).

Exibir data do artigo

Defina como SIM para exibir publicamente a data de envio do artigo da base de conhecimento (a data é sempre vísivel no painel de controle da equipe).

Máximo de resultados na pesquisa

Este é o máximo de resultados que o Hesk retornará quando alguém pesquisar a base de conhecimento.

Tamanho da visualização do artigo

Tamanho máximo (número de caracteres) do conteúdo do artigo da base de conhecimento que será exibido na lista de artigos ao navegar nas categorias da base de conhecimento.

Exibir artigos populares

Número de artigos populares (mais visitados) exibidos nas páginas principais da central de suporte e da base de conhecimento. Defina como 0 para desabilitar a listagem de artigos populares.

Exibir últimos artigos

Número dos últimos artigos (enviados mais recentemente) exibidos nas páginas principais da central de suporte e da base de conhecimento. Defina como 0 para desabilitar a listagem de últimos artigos.
