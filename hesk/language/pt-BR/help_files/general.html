Ajuda interativa para as configurações do Hesk

 

» Configurações gerais

Título do site

Um título curto do seu site, por exemplo "Meu amável site". Isto será usado como o título do link que é exibido no canto superior esquerdo da área do cliente para permitir a eles que retornem ao seu site.

URL do site

URL do seu site, por exemplo "http://www.seusite.com.br". Certifique-se de que esta URL é válida e que você a iniciou com http:// ou https:// . Isto será usado como a URL do link que é exibido no canto superior esquerdo da área do cliente e permite a eles que retornem ao seu site.

E-mail do administrador

Endereço de e-mail padrão do administrador. Certifique-se de digitar um endereço de e-mail válido, pode ser o mesmo que o e-mail do suporte.

"De:" e-mail

Seu endereço de e-mail exibido no campo "De:". Este é o endereço de e-mail que será exibido como o remetente do e-mail (cabeçalho De:) em todos os e-mails enviados pelo HESK. Geralmente é um e-mail que você não lê e não responde. Certifique-se de digitar um endereço de e-mail válido, pode ser o mesmo que o e-mail do suporte, mas é recomendável usar outro.

"De:" nome

Este é o nome que será exibido como remetente do e-mail (De:) em todos os e-mails enviados pelo HESK. Se nenhum nome for definido o cabeçalho De: do e-mail irá exibir somente o endereço de e-mail definido em "De: e-mail".

» Configurações de idioma

Idioma padrão

Se você possuir vários idiomas instalaods no HESK este será o idioma padrão. Todos os idiomas instalados corretamente são exibidos na caixa de seleção. Por favor consulte o arquivo readme.html para obter instruções sobre como instalar mais idiomas. O link "Testar pasta de idiomas (language)" o ajudará a solucionar problemas de instalações de idiomas não funcionando.

Se múltiplos idiomas estão habilitados você pode definir seu idioma preferencial na página de "Perfil".

Múltiplos idiomas

Habilitar esta configuração permitirá que seus usuários e equipe escolham seu idioma preferido da lista de idiomas instalados. Não habilite este recurso a menos que você forneça suporte em todos os idiomas instalados!

» Configurações do banco de dados

Servidor do banco de dados

Servidor do banco de dados, "localhost" funciona para a maioria dos usuários. NÃO altere isto a menos que você saiba qual é o seu servidor MySQL. Pergunte a sua empresa de hospedagem se não tiver certeza ou se tiver problemas ao se conectar ao banco de dados MySQL.

Nome do banco de dados

O nome de seu banco de dados MySQL. Você deve criar um no painel de controle de sua hospedagem web ou solicitar a sua empresa de hospedagem esta informação.

Usuário do banco de dados

O usuário que você usa para se conectar ao seu banco de dados MySQL. Você deve criar um usuário MySQL no painel de controle de sua hospedagem web ou solicitar a sua empresa de hospedagem por esta informação.

 

Fechar janela

 

Senha do banco de dados

A senha que você usa para se conectar ao banco de dados MySQL. Você deve criar um usuário MySQL no painel de controle de sua hospedagem web ou solicitar a sua empresa de hospedagem por esta informação.

Prefixo da tabela

O prefixo que será adicionado em todos os nomes de tabelas. Se definido como hesk_ a tabela "users" será na verdade nomeada como "hesk_users" em seu banco de dados. Isto é útil se você desejar rodar múltiplas instalações do HESK usando um banco de dados MySQL, apenas dê a cada instalação um nome de prefixo de tabela único (hesk1_, hesk2_, hesk3_, ...)
